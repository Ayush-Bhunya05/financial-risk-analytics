# 📈 Financial Risk Analytics

An end-to-end Business Intelligence project that analyzes large-scale financial transaction data using **Python**, **SQL**, **Microsoft Excel**, and **Power BI** to identify fraud patterns, monitor financial risk indicators, and generate actionable business insights.

---

## 🎯 Project Objectives

- Analyze financial transaction data
- Monitor fraud trends and risk indicators
- Perform customer and merchant risk analysis
- Generate interactive dashboards for business users
- Build data-driven insights for decision making

---

## 🛠 Tech Stack

| Technology | Purpose |
|------------|---------|
| Python (Pandas, NumPy) | Data Cleaning & Feature Engineering |
| SQL | Data Analysis & Business Queries |
| Microsoft Excel | Data Preparation & Analysis |
| Power BI | Dashboard & Data Visualization |
